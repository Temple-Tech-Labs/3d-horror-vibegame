# 3d-horror-vibegame

A spooky-not-scary 3D third-person browser adventure built with Three.js and Vite, playable on both desktop and Samsung Galaxy Tab S10 FE. Inspired by the family-friendly tone of Luigi's Mansion and Hotel Transylvania, designed for ages 8–15 — more "haunted funhouse" than "survival horror". The game runs entirely in the browser with no installs required, using a virtual joystick on touch devices (nipplejs) and WASD/arrow keys on desktop.

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start the local development server |
| `npm run build` | Build the production bundle into `dist/` |
| `npm run deploy` | Build and publish to GitHub Pages |
